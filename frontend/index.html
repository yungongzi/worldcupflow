技术架构与训练数据
📈

双模型架构

XGBoost 分类器 预测胜平负（3分类），XGBoost Poisson 回归 预测进球数，泊松概率矩阵计算比分概率。

⚙️

特征工程

39 维特征：Elo等级分（带赛事权重）、近5/10场状态、对战记录、攻防能力、赛事重要性、比赛间隔等。

🌐

数据策略

混合时间策略：Elo用1872-2025全历史数据积累稳定性，ML模型用2002-2025现代足球数据训练，反映当代战术风格。

🔄

实时更新

ESPN API 抓取实时比赛，自动归一化队名，定时刷新预测，支持后台触发模型重训。

重要提示 / Disclaimer

预测结果仅供参考，足球比赛存在高度不确定性，请理性看待预测结果。

本系统基于历史统计模型（XGBoost + Poisson）进行概率预测，未考虑以下不可量化的实时因素：

  • 🤕 球员伤病：核心球员伤停可能显著影响比赛走势
  • 🟥 红黄牌：比赛中红牌导致的少打多情况无法预判
  • ⏱️ 伤停补时：补时阶段的进球具有随机性
  • 🌧️ 天气与场地：雨雪、高温、海拔等环境因素
  • 👔 临场战术：教练排兵布阵、轮换、临场调整
  • 🧠 心理因素：球员压力、士气、更衣室状态
  • VAR 视频裁判：关键判罚可能改变比赛结果

历史数据不能完全代表未来，任何预测都无法保证 100% 准确。 请将本工具作为辅助参考，而非投注或决策的唯一依据。
